# Nx Workspaces Playground Project Brief

## Project Overview
- **Name**: Nx Playground
- **Version**: 1.0.0
- **Description**: Nx Workspaces Playground for the course by Decoded Frontend
- **Author**: Dmytro Mezhenskyi

## Technical Requirements
- Node.js ≥ 22.0.0
- npm ≥ 10.0.0

## Core Dependencies
- Express.js for server functionality
- TypeScript for type-safe development
- Jest for testing
- ESLint and Prettier for code quality

## Project Goals
- Provide a playground environment for learning Nx workspaces
- Demonstrate best practices in Angular/Nx development
- Serve as a foundation for the Decoded Frontend course

## Development Standards
- TypeScript for all development
- Jest for testing with coverage reporting
- ESLint and Prettier for consistent code style
- Comprehensive documentation in memory-bank structure
